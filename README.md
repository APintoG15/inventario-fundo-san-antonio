# 📦 Fundo San Antonio - Control de Inventarios + Facturación

Sistema integral de gestión de inventarios, facturación y análisis de KPIs para Fundo San Antonio.

## ✨ Características

- 📈 **Dashboard completo** con 8 KPIs principales
- 🔍 **Scanner de códigos** EAN-13/UPC en tiempo real
- 🛒 **Generador de facturas** con PDF automático (Contasis compatible)
- 📊 **Gráficos interactivos** de ventas, productos y categorías
- 💾 **Almacenamiento local** (localStorage) - tus datos privados
- 🎨 **Identidad visual** de Fundo San Antonio (colores corporativos)

## 🚀 Cómo usar

1. **Descarga** el archivo `inventory_fundo_san_antonio.html`
2. **Abre** en tu navegador (Chrome, Firefox, Safari, Edge)
3. **No necesita internet** después de cargar
4. Los datos se guardan automáticamente en tu navegador

## 📊 KPIs Disponibles

- Productos registrados
- Stock total
- Valor total del inventario
- Movimientos registrados
- Facturas emitidas
- Ingresos totales
- Ventas últimos 7 días
- Stock en riesgo (alertas)

## 📱 Tabs Principales

1. **📈 Dashboard** - Vista general con gráficos
2. **🔍 Scanner** - Escanear códigos entrada/salida
3. **🛒 Factura** - Crear facturas con PDF
4. **📋 Productos** - Gestionar catálogo
5. **📊 Análisis** - Detalles por categoría y producto

## 🎨 Identidad Visual

Diseñado con los colores corporativos de Fundo San Antonio:

## 💡 Hardware Recomendado

**Scanner:** Zebra DS3678 Bluetooth ($80-120)
**CPU:** Mini PC o Laptop vieja con WiFi

## 📄 PDFs Generados

Las facturas se generan automáticamente con:
- Código de producto
- Nombre y cantidad
- Precio unitario
- IVA (19%)
- Total a pagar
- Branding Fundo San Antonio

Compatible con Contasis para importación de datos.

## 🔒 Privacidad

- ✅ Todos los datos se guardan **localmente** en tu navegador
- ✅ **Ningún dato** se envía a servidores
- ✅ **Completamente privado** y seguro

## ⚙️ Requisitos

- Un archivo HTML (no necesita instalación)
